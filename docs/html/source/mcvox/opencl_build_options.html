OpenCL build optionsÂ¶

OpenCL build options can be used to further optimize the performance of the simulator kernel. These options can be passed to the constructor xopto.mcvox.mc.Mc.__init__() by populating the cl_build_options argument wit a list of OpenCL build options. The build options can be passed as strings defined by the OpenCL standard <https://www.khronos.org/registry/OpenCL/sdk/1.0/docs/man/xhtml/clBuildProgram.html> or by using predefined options from the xopto.cl.cloptions.

The build options are conveniently imported into the xopto.mcvox.mc as cloptions.

For a full list of options see xopto.cl.cloptions

Note

Note that the most significant performance improvment is usually obtained by using the '-cl-fast-relaxed-math' (FastRelaxedMath) build option.`
